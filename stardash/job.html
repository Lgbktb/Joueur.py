Job¶

class games.stardash.job.Job¶

Bases: games.stardash.game_object.GameObject

The class representing the Job in the Stardash game.

Information about a unit’s job.

carry_limit¶

How many combined resources a unit with this Job can hold at once.

Return type

int

damage¶

The amount of damage this Job does per attack.

Return type

int

energy¶

The amount of starting health this Job has.

Return type

int

game_object_name¶

String representing the top level Class that this game object is an instance of. Used for reflection to create new instances on clients, but exposed for convenience should AIs want this data.

Return type

str

id¶

A unique id for each instance of a GameObject or a sub class. Used for client and server communication. Should never change value after being set.

Return type

str

log(message)¶

Adds a message to this GameObject’s logs. Intended for your own debugging purposes, as strings stored here are saved in the gamelog.

Parameters

message (str) – A string to add to this GameObject’s log. Intended for debugging.

logs¶

Any strings logged will be stored here. Intended for debugging.

Return type

list[str]

moves¶

The distance this job can move per turn.

Return type

int

range¶

The distance at which this job can effect things.

Return type

int

shield¶

The reserve the martyr use to protect allies.

Return type

int

title¶

The Job title. ‘corvette’, ‘missileboat’, ‘martyr’, ‘transport’, or ‘miner’. (in this order from 0-4).

Return type

str

unit_cost¶

How much money it costs to spawn a unit.

Return type

int
