Structure¶

class games.catastrophe.structure.Structure¶

Bases: games.catastrophe.game_object.GameObject

The class representing the Structure in the Catastrophe game.

A structure on a Tile.

effect_radius¶

The range of this Structure’s effect. For example, a radius of 1 means this Structure affects a 3x3 square centered on this Structure.

Return type

int

game_object_name¶

String representing the top level Class that this game object is an instance of. Used for reflection to create new instances on clients, but exposed for convenience should AIs want this data.

Return type

str

id¶

A unique id for each instance of a GameObject or a sub class. Used for client and server communication. Should never change value after being set.

Return type

str

log(message)¶

Adds a message to this GameObject’s logs. Intended for your own debugging purposes, as strings stored here are saved in the gamelog.

Parameters

message (str) – A string to add to this GameObject’s log. Intended for debugging.

logs¶

Any strings logged will be stored here. Intended for debugging.

Return type

list[str]

materials¶

The number of materials in this Structure. Once this number reaches 0, this Structure is destroyed.

Return type

int

owner¶

The owner of this Structure if any, otherwise None.

Return type

games.catastrophe.player.Player

tile¶

The Tile this Structure is on.

Return type

games.catastrophe.tile.Tile

type¶

The type of Structure this is (‘shelter’, ‘monument’, ‘wall’, ‘road’, ‘neutral’).

Return type

str
