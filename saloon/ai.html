find_path(start, goal)¶
A very basic path finding algorithm (Breadth First Search) that when

given a starting Tile, will return a valid path to the goal Tile.

Parameters

  • start (games.saloon.tile.Tile) – the starting Tile

  • goal (games.saloon.tile.Tile) – the goal Tile

Returns

A list of Tiles representing the path, the the first element being a valid adjacent Tile to the start, and the last element being the goal.

Return type

list[games.saloon.tile.Tile]

get_setting(key)¶

Gets an AI setting passed to the program via the –aiSettings flag. If the flag was set it will be returned as a string value, None otherwise.

Parameters

key (str) – The key of the setting you wish to get the value for

Returns

A string representing the value set via command line, or None if the key was not set

Return type

str

This is called every time it is this AI.player’s turn.

Represents if you want to end your turn. True means end your turn, False means to keep your turn going and re-call this function.
